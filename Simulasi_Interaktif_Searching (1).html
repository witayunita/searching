📌 Apa itu Linear Search?

Linear Search adalah algoritma pencarian yang memeriksa setiap elemen dalam array satu per satu dari awal sampai akhir hingga menemukan nilai yang dicari.

⚡ Apa itu Binary Search?

Binary Search adalah algoritma pencarian yang membagi array menjadi dua bagian secara berulang hingga menemukan nilai yang dicari. ⚠️ SYARAT: Data harus TERURUT!

📊 Perbandingan Linear Search vs Binary Search

Lihat perbedaan performa antara kedua algoritma pada berbagai ukuran data!

Aspek Linear Search Binary Search
Cara Kerja Memeriksa satu per satu dari awal sampai akhir Membagi array menjadi dua bagian berulang kali
Syarat Data ✅ Boleh acak (tidak harus terurut) ⚠️ HARUS terurut (ascending/descending)
Kecepatan 🐢 Lambat untuk data besar 🚀 Sangat cepat bahkan untuk data besar
Kompleksitas Waktu (Terburuk) O(n) - Proporsional dengan jumlah data O(log n) - Logaritmik
Contoh: 10 data Maksimal 10 langkah Maksimal 4 langkah
Contoh: 100 data Maksimal 100 langkah Maksimal 7 langkah
Contoh: 1,000 data Maksimal 1,000 langkah Maksimal 10 langkah
Contoh: 1,000,000 data Maksimal 1,000,000 langkah Maksimal 20 langkah
Kapan Digunakan? • Data kecil
• Data tidak terurut
• Pencarian sekali saja		 • Data besar
• Data sudah terurut
• Pencarian berulang kali

💡 Kesimpulan

Binary Search jauh lebih efisien dari Linear Search, terutama untuk data besar. Namun, Binary Search HANYA bisa digunakan pada data yang sudah terurut. Jika data tidak terurut dan tidak mungkin diurutkan, gunakan Linear Search.
